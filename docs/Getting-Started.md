# Getting Started

## First Launch

When NAVFIT26 opens, you land on the **Home Page**, which lists all databases currently tracked by the application.

---

## Creating a New Database

1. Click **+ New Database** on the Home Page.
2. A Save dialog opens — enter a filename and choose a save location. The default is `MyEvals.db` on your Desktop.
3. NAVFIT26 creates a new SQLite database from the built-in template and adds it to your list.

---

## Opening an Existing Database

- **From the list**: Click a database card on the Home Page to open it.
- **Open a file**: Click **Open Database** to import an existing file. Accepted formats: `.db`, `.sqlite`, and `.accdb`.

> If you select a NAVFIT98 `.accdb` file, NAVFIT26 will automatically convert it to a SQLite database and add it to the list.

---

## Navigating the App

| Area | How to reach it |
|---|---|
| Home Page | Click **← Back** from the opened database view, or use the menu to navigate home |
| FITREP list | Click a database card on the Home Page |
| FITREP form | Click **+ Add Report** or **Edit / View** on an existing report |
| Export options | Inside the FITREP form — **Export PDF** and **Export ACCDB** buttons |

---

## Returning to the Home Page

From the opened database view, click **← Back**. From within a FITREP form, click **← Back to Database**. If you have unsaved changes, a confirmation prompt will appear.

---

## Deleting a FITREP

In the opened database view, click **Delete** next to a report. A confirmation prompt appears before the record is permanently removed.

> To remove a database from the tracked list without deleting the file on disk, click the **✖** button on its database card.
